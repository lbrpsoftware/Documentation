# Books

## 1. Inleiding

Met de Books App kun je e-books aankopen, lezen en beheren. Gebruikers kunnen boeken in verschillende talen bekijken, aankopen en lezen. Tijdens het lezen is het mogelijk om vragen te stellen aan een AI-model over de inhoud van het boek. Beheerders kunnen nieuwe boeken toevoegen en bestaande boeken beheren.
