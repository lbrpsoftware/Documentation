# LBRP Cloud - Identity - Facturen

## 1. Facturen

Op het tabblad **Facturen** kun je je maandelijkse kostenoverzicht bekijken en beheren:

1. **Maandelijkse facturen**
   - Elke maand wordt automatisch een factuur gegenereerd op basis van het gebruik van de geactiveerde applicaties.

2. **Details bekijken**
   - Klik op een factuur om de specificaties te bekijken, waaronder het gebruik en de kosten per applicatie.

3. **Factuur betalen**
   - Betaal de openstaande factuur eenvoudig via de beschikbare betaalmethoden (**Stripe** of **PayPal**).

Dit tabblad biedt een duidelijk overzicht van je uitgaven en maakt het eenvoudig om betalingen bij te houden.
